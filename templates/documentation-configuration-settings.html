Einrichtung: Systemeinstellungen

Einige Features des Protokollsystem müssen bei der Installation und Einrichtung durch die Admins aktiviert werden. Eine Übersicht über die mit diesen Systemeinstellungen verbundenen Features soll hier gegeben werden. Sollten einige der angegeben Funktionalitäten aktiviert werden, so wende dich an die Admins. Bedenke vor deiner Anfrage, dass einige Features eventuell aus wichtigen Gründen bisher noch nicht aktiviert wurden.

Feature Aktiviert Kurzbeschreibung
E-Mailversand Das System kann selbstständig E-Mails versenden, um über nächste Sitzungen, neue Protokolle oder Todos zu informieren.
Druckeranbindung Das System kann Druckaufträge starten, um Beschlüsse oder Protokolle direkt über das Webinterface zu drucken.
Etherpadanbindung Das System kann mit einem Etherpad interagieren, um das Schreiben von Protokollen dort zu ermöglichen.
Wikianbindung Das System kann veröffentlichte Protokolle in ein Wiki übertragen.
Kalenderanbindung Das System kann eingetragene Sitzungstermine in einen Kalender übertragen.
Angepasste Standardvorlage Das System nutzt die Standardprotokollvorlage in einer angepassten Form.
Individuelle Protokollvorlagen Das System stellt zusätzlich zur Standardvorlage individuelle Protokollvorlagen zur Verfügung.
