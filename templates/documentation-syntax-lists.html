Listen

Liste

Die Liste ist das einzige strukturierende Element. Listen können beliebig geschachtelt werden. Die Elemente der Liste werden durch das Zeilenende getrennt. 
{TOP Tagesordnungspunkt
    Zeile mit Text drin
    Hier eine geschachtelte Liste: {
        Details zu diesem Punkt.
        Mehr Details zu diesen Punkt.
    }
}
Text, der vor der öffnenden Klammer { steht, wird als Name der Liste angesehen, nur TOPs können Text hinter der öffnenden Klammer haben. Listen in der höchsten Ebene müssen TOPs sein. {% endblock %}
